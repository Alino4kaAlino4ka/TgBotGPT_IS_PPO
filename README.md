# Telegram-бот для дисциплины «Информационные системы» 

Интеллектуальный помощник для студентов направления 44.03.02 Психолого-педагогическое образование, изучающих дисциплину «Информационные системы». Бот содержит локальную базу знаний по основным темам курса и интегрирован с языковой моделью для генерации ответов на дополнительные вопросы, отсутствующие в базе.

## Функциональные возможности

Бот предоставляет студентам и преподавателям следующие возможности:

1. Получение определений и пояснений по ключевым терминам дисциплины: информационная система, информационные технологии, HTML, интеллект-карта, инфографика и другие.

2. Ответы на типовые вопросы по темам рабочей программы: создание веб-страниц, применение интеллект-карт, образовательные ресурсы интернета, корреляционный анализ.

3. Обращение к языковой модели для генерации ответов на дополнительные вопросы, отсутствующие в локальной базе знаний.

4. Быстрая навигация по курсу с помощью команд: /start и /help.

5. Безопасное использование благодаря системе фильтрации нежелательных запросов.

## Установка и запуск

Для запуска бота необходимо выполнить следующие шаги:

1. Открыть файл `bot.ipynb` в Google Colab.

2. Создать бота в Telegram через @BotFather и получить токен.

3. Получить API-ключ для VseGPT на сайте https://api.vsegpt.ru.

4. Добавить токены в секреты Colab:
   - `TELEGRAM_BOT_TOKEN`
   - `VSEGPT_API_KEY`

5. Запустить ячейки ноутбука последовательно.

6. Открыть Telegram и начать диалог с ботом.

## База знаний

База знаний бота сформирована на основе рабочей программы дисциплины «Информационные системы» и включает записи по всем темам курса: понятия ИТ и ИС (информационная система, информационные технологии, глобальная информационная технология); создание web-интерфейса (HTML, алгоритм создания веб-страницы, конструкторы сайтов, гипертекст); интеллект-карты и инфографика (законы построения, онлайн-сервисы, применение в образовании); информационные ресурсы интернета (образовательные ресурсы, браузер, база данных, сервисы интернета, электронная почта); корреляционный анализ (коэффициент Пирсона, критерий Стьюдента, критерий Манна-Уитни); глоссарий (антивирус, архиватор, драйвер, модем, провайдер, сканер). Каждая запись содержит структурированный ответ с примерами из педагогической и психологической практики.

## Примеры использования

### Пример 1. Получение определения из базы знаний

Запрос пользователя:
Что такое информационная система?



Ответ бота:
Информационная система (ИС) — это система, предназначенная для хранения, поиска и обработки информации, а также соответствующие организационные ресурсы (человеческие, технические, финансовые), которые обеспечивают и распространяют информацию

Состав ИС включает:
• техническое обеспечение (аппаратные средства)
• программное обеспечение
• информационное обеспечение (базы данных)
• организационное обеспечение (регламенты, инструкции)
• кадровое обеспечение (специалисты)



### Пример 2. Генерация ответа через языковую модель

Запрос пользователя:
Как применение информационных систем влияет на эффективность психологического консультирования?



Ответ бота (генерируется через VseGPT API с учётом специализации студента):
Применение информационных систем в психологическом консультировании позволяет автоматизировать ведение документации, ускорить доступ к результатам диагностики и организовать дистанционное взаимодействие с клиентами. Специализированные системы для психологов предоставляют шаблоны заключений, каталоги методик и инструменты для обработки статистических данных, что повышает точность выводов и экономит время консультанта.

 

### Пример 3. Использование системного промпта

При обращении к языковой модели бот использует системный промпт, который учитывает специфику дисциплины и аудитории:
Ты — ассистент преподавателя Алины Сергеевны по дисциплине «Информационные системы» для студентов-педагогов-психологов. Отвечай кратко, структурированно и понятно. Используй примеры из образования и психологии. Если вопрос не по теме, вежливо направь к учебным материалам. НЕ используй markdown-разметку, только обычный текст с эмодзи для визуального оформления.



### Пример 4. Безопасность

При обнаружении нежелательного запроса бот отвечает отказом:

Запрос пользователя:
как взломать сайт?



Ответ бота:
Извините, я не могу помочь с этим вопросом. Я создан для помощи в изучении информационных систем в образовательных целях. Если у вас есть вопросы по учебному материалу, я с радостью отвечу!



## Планы развития

В перспективе планируется расширение функциональности бота по следующим направлениям:

1. Добавление системы регистрации студентов для персонализации взаимодействия и отслеживания индивидуального прогресса.

2. Сбор и анализ статистики вопросов для выявления тем, вызывающих наибольшие затруднения у студентов, и последующей корректировки учебного процесса.

3. Интеграция с платформой дистанционного обучения РосНОУ (Moodle) для автоматического обмена данными об активности студентов.

4. Расширение базы знаний по дополнительным темам дисциплины на основе анализа студенческих запросов и включения новых разделов учебного курса.

5. Внедрение адаптивной системы подбора сложности ответов в зависимости от уровня подготовки студента, что позволит индивидуализировать процесс обучения.

6. Создание веб-интерфейса для редактирования базы знаний преподавателем без необходимости изменения программного кода.

7. Внедрение RAG-системы для работы с PDF-документами и учебными пособиями, что позволит боту обрабатывать большие объёмы текстовой информации.

8. Интеграция локальной трансформерной модели для автономной генерации ответов без обращения к внешним API.

9. Масштабирование решения на другие дисциплины факультета педагогики и психологии.

10. Разработка полноценного мобильного приложения для удобства использования бота.
